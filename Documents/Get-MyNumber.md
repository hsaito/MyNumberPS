# Get-MyNumber

## SYNOPSIS
Get randomly generated MyNumber.

## SYNTAX

```
Get-MyNumber [<CommonParameters>]
```

## DESCRIPTION
This command generates randomized MyNumber for testing. Result is generated as MyNumberData which capsulates integer array.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-MyNumber
```

Generates Mynumber
